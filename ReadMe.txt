YouFace social network web application
Allows user to create own account, custumize his or her account, add photo to profile
Allows to edit profile information
Allows user to post new posts, like own and other users' posts
Allows user to search for new friends among all users or using search bar
Allows user to add new friand after his or her new friend confirmed friendship
Allows user to like posts of other users

For better user experience create many new accounts and log in / log out among them, add new friends, like their posts, everything you like!

(To change user's information, enter his or her actual password, else information won't be saved!)

Have fun using YouFace
